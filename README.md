## react 專案開啟方式

1. 將 food-dlv-website 資料夾以 vscode 打開後，開啟終端機 (預設快捷鍵 ctrl+shift+`)
2. 輸入指令 `npm install` (跟 nuget 套件一樣，這邊的目的是要安裝 react 所需的 nodejs npm 套件)
3. 跑完之後，輸入 `npm start` ，就可以打開了~

各API 皆已失效~
